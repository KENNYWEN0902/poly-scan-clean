PolymarketBot 需求文档

更新时间：2026-04-24

正式本地项目目录：/Users/kennywen/Documents/Playground/polymarketbot

运行目标

  • 连接 Polymarket RTDS 获取 BTC/USD Chainlink 价格。
  • 使用 pUSD/CLOB collateral 作为 Dashboard 可用余额来源。
  • Dashboard 稳定展示账户、策略、持仓、历史、设置和 WebSocket 在线状态。

功能需求

  • RTDS 使用当前订阅协议并保持 PING 保活，断线自动重连。
  • /api/account 优先查询 CLOB collateral/pUSD，并带短超时和缓存。
  • /api/trades 没有交易时返回 []。
  • Dashboard 任意单个 API 慢或失败时，页面主体仍应显示。
  • Dashboard WebSocket 由全局 App 管理，页面组件只订阅事件。

验收标准

  • PM2 中 poly-bot-api、poly-bot-btc、poly-dashboard 均为 online。
  • 日志出现 RTDS BTC/USD price established。
  • Dashboard 左下角显示正确余额和“在线 · 已同步”。
  • 历史页无交易时显示“暂无交易记录”。
