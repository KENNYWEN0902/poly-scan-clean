一、最推荐的做法

  1. 本地只修改非敏感配置。
  2. 本地不要保存真实私钥。
  3. 上传到服务器后，再创建真正的 .env。
  4. 编译和运行都放在 Ubuntu 服务器完成。
私钥、API Secret、Passphrase 这类敏感值，不建议提前写进本地项目目录，尤其是 Documents 目录可能会被同步或备份。

二、这个 bot 最关键的 3 个参数

POLY_PRIVATE_KEY=0x你的浏览器钱包私钥
POLY_SIGNATURE_TYPE=2
POLY_FUNDER_ADDRESS=0x你在Polymarket个人资料页看到的地址

其中：

  • POLY_PRIVATE_KEY：建议只在服务器上填写。
  • POLY_SIGNATURE_TYPE=2：适合你现在这条浏览器钱包路线。
  • POLY_FUNDER_ADDRESS：用 Polymarket 个人资料页地址，不是充值弹窗地址。

三、本地应该改什么

我给你准备了一份安全模板文件：env.server.template。

你可以在本地先填写这些非敏感内容：

  • POLY_SIGNATURE_TYPE=2
  • POLY_FUNDER_ADDRESS=你的个人资料页地址
  • POLY_TRADE_AMOUNT=5
  • API_PORT=9876

但先不要在本地填写真实 POLY_PRIVATE_KEY。

cd /Users/kennywen/Documents/Playground/polymarketbot
cat env.server.template
nano env.server.template

四、上传到服务器

方式 A：命令行上传

如果你用的是 ubuntu 用户：

方式 B：图形工具上传

你也可以用 FileZilla、Cyberduck、FinalShell。服务器目标路径建议固定为 /root/poly-scan，这样就不用改 PM2 配置里的路径。

五、服务器上如何配置

ssh root@你的服务器IP
cd /root/poly-scan
cp env.server.template .env
nano .env

把模板中的私钥占位符改成真实私钥，然后确认下面两项正确：

POLY_SIGNATURE_TYPE=2
POLY_FUNDER_ADDRESS=0x你个人资料页中的地址

一个适合你现在的小额测试版本：

POLY_PRIVATE_KEY=0x你的真实钱包私钥
POLY_SIGNATURE_TYPE=2
POLY_FUNDER_ADDRESS=0x你个人资料页中的地址
POLY_TAKE_PROFIT_PCT=0.50
POLY_STOP_LOSS_PCT=0.30
POLY_TRADE_AMOUNT=5
API_PORT=9876

十、如果你想从浏览器访问 dashboard

nano /etc/nginx/sites-available/poly-bot
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:3457;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:9876;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /ws {
        proxy_pass http://127.0.0.1:9876/ws;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
ln -sf /etc/nginx/sites-available/poly-bot /etc/nginx/sites-enabled/poly-bot
rm -f /etc/nginx/sites-enabled/default
nginx -t
systemctl restart nginx

然后浏览器访问：http://你的服务器IP

最后一句最重要

POLY_FUNDER_ADDRESS 用个人资料页地址；充值 用充值弹窗地址；私钥 只在服务器上填写。
